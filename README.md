# Experimental Oceanography Group Website

This is the website for the Experimental Oceanography research group.

## Built with Hugo Research Group Theme

This site uses the [Hugo Research Group Theme](https://github.com/wowchemy/starter-hugo-research-group) powered by [Wowchemy](https://hugoblox.com) and [Hugo](https://github.com/gohugoio/hugo).

## Theme Features

- Homepage with group overview, news, publications, events, and team profiles
- Support for technical content with Markdown, LaTeX math, diagrams, and Jupyter
- BibTeX publication import
- Customizable widgets and themes
